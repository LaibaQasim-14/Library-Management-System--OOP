# Library Management System

A console-based Library Management System developed in C++ using Object-Oriented Programming (OOP) concepts. The project automates core library operations including user management, resource handling, borrowing, fines, reservations, notifications, analytics, and report generation.

## Features
- Member & Admin authentication
- Multiple admin roles with permission hierarchy
- Role-based access control
- Book & Ebook management
- Borrow & return system
- Automatic fine calculation and overdue detection
- Membership tier system (Silver, Gold, Platinum)
- Borrow extension request management
- Reservation & waitlist system
- Real-time resource availability tracking
- Book rating and review system
- Notifications & report generation
- Analytics dashboard for resource usage and member activity
- Search by title, author, category, or ISBN
- Account suspension for excessive fines
- Audit logging for admin activities
- Password hashing, input validation, and account lockout handling
- File-based persistent data storage
- Modular architecture using centralized LibraryManager

## OOP Concepts Used
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Exception Handling
- Composition & Aggregation
- Operator Overloading
- File Handling
